# Learning Nest.js

Diving into the Nest.js, Typescript, Node.js, Postgresql, TypeORM, Express.js and REST Backend.

This project was created using the nest.js cli

### Installation

```bash
$ npm install
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
